# MAKA Accesorios – App E-commerce

Proyecto desarrollado para la materia Proyecto II.
Sprint 1 - MVP funcional.

Funcionalidades:
- Catálogo
- Detalle de producto
- Carrito dinámico
- Checkout visual

Tecnologías:
- React
- Vite
- React Router
- Context API

REPORTE DE SPRINTS 
1. Fases de los Sprint
Sprint 1 – Base del sistema

Fase de planificación:
Definición de la estructura del proyecto
Organización de carpetas (components, pages, context, data)

Fase de desarrollo:
Creación de navegación principal
Implementación del catálogo de productos
Desarrollo de páginas:
Home
Detalle del producto
Carrito
Checkout
Lógica inicial del carrito con Context API

Fase de revisión y ajustes:
Corrección de identidad visual (color #c18e61)
Ajuste de tipografías (Bile y Montserrat)
Estandarización de imágenes (2048x2048 px)

Fase de entrega:
Despliegue inicial funcional

Sprint 2 – Mejora funcional

Fase de planificación:
Enfoque en experiencia de usuario y funcionalidad del e-commerce

Fase de desarrollo:
Mejora del carrito (agregar, visualizar, checkout)
Optimización de rutas con React Router
Consolidación de datos (products.js)
Mejora visual del catálogo

Fase de revisión y ajustes:
Migración de React Scripts a Vite
Corrección de errores de build
Organización del código
Solución de problemas con GitHub

Fase de entrega:
Despliegue funcional en Vercel
Aplicación estable y navegable

Sprint 3 – Integración total

Fase de planificación:
Integrar todos los módulos del sistema

Fase de desarrollo:
Flujo completo:
Catálogo → Detalle → Carrito → Checkout
Mejora de usabilidad e interfaz

Fase de revisión y ajustes:
Corrección de errores:
Rutas
Actualización del carrito
Visualización de componentes
Optimización del diseño
Limpieza y reorganización del código

Fase de entrega:
App completamente funcional
Despliegue en producción
Documentación (README)
